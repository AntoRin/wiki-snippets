Search for a Wikipedia page

API
You can use the website's interface directly to search for and get snippets of Wikipedia information. If you choose to use the snippets in your application instead, you can use the API at the path "/api/wikisnippet/:query".

Query Structure:
  • Although spaces between words are usually encoded by the browser, it is recommended to encode spaces with "%20" or with a "+".
  • Using search queries that are not understood by Wikipedia will return an error.
  • Searching for exact Wikipedia pages will give you an accurate result.
